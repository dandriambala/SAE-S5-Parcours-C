Étude : Consommation de jeux vidéo et performances académiques

Notre groupe avait pour but de recueillir des données afin d’analyser le lien entre consommation de jeux vidéo et performances académiques. Pour cela, un questionnaire a été conçu, permettant de récolter 269 réponses d’étudiants français en études supérieures.

Notre groupe avait pour but de recueillir des données afin d’analyser le lien entre consommation de jeux vidéo et performances académiques. Pour cela, un questionnaire a été conçu, permettant de récolter 269 réponses d’étudiants français en études supérieures. Les étudiants provenaient de divers établissements, dans des domaines d’étude variés, bien qu’une sur-représentation des étudiants en informatique ait été constatée.

Informations sociodémographiques

Les données obtenues contiennent des informations sociodémographiques : genre, âge, niveau et domaine d’étude, niveau d’étude parental, situation familiale, etc.

Consommation de jeux vidéo

Nous avons également récolté des données concernant la consommation de jeux vidéo : quantité d’heures jouées, types de jeux, moment de jeu dans la journée, langue dans laquelle les individus jouent, etc.

Activités du quotidien

Afin de relativiser l’impact des jeux vidéo sur les performances scolaires, nous avons également relevé des données concernant différentes activités du quotidien : loisirs en général, temps consacrés aux tâches ménagères, temps consacré aux réseaux sociaux, etc.

Performances académiques

Enfin, notre jeu de données contient des informations permettant de refléter la performance académique : moyenne générale, moyenne en anglais. De même que dans l’étude Student Alcohol Consumption, nous avons relevé des facteurs externes pouvant influencer cette performance : retards, absences, motivation à poursuivre des études, temps de révision, temps de trajet, soutien scolaire, etc.
