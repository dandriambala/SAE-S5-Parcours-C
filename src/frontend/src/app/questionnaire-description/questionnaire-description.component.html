Analyse du questionnaire

Notre groupe avait pour but de recueillir des données afin d’analyser le lien entre consommation de jeux vidéo et performances académiques. Pour cela, un questionnaire a été conçu, permettant de récolter 269 réponses d’étudiants français en études supérieures. Les étudiants provenaient de divers établissements, dans des domaines d’étude variés, bien qu’une sur-représentation des étudiants en informatique ait été constatée.

Informations sociodémographiques

Les données obtenues contiennent des informations suivantes :

  • Genre, âge, niveau et domaine d’étude
  • Niveau d’étude parental
  • Situation familiale

Consommation de jeux vidéo

Nous avons également récolté des données concernant la consommation de jeux vidéo, incluant :

  • Quantité d’heures jouées
  • Types de jeux
  • Moment de jeu dans la journée
  • Langue dans laquelle les individus jouent

Activités quotidiennes et performances académiques

Afin de relativiser l’impact des jeux vidéo sur les performances scolaires, nous avons également relevé des données sur :

  • Loisirs en général
  • Temps consacré aux tâches ménagères
  • Temps consacré aux réseaux sociaux

Enfin, notre jeu de données contient des informations permettant de refléter la performance académique : moyenne générale, moyenne en anglais. De même que dans l’étude Student Alcohol Consumption, nous avons relevé des facteurs externes pouvant influencer cette performance :

  • Retards
  • Absences
  • Motivation à poursuivre des études
  • Temps de révision
  • Temps de trajet
  • Soutien scolaire
