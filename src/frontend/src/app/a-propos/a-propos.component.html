Origine du logo et de notre nom d'équipe

Le nom de couleur rose Mountbatten désigne une couleur de camouflage maritime utilisée pour les navires de la Royal Navy au cours de la Seconde Guerre mondiale.
C'était un gris rosâtre, employé à l'instigation de l'amiral Louis Mountbatten. Mountbatten, convaincu de l'efficacité de cette couleur comme camouflage à l'aube et au crépuscule, moments dangereux pour les navires,
fit peindre tous les destroyers de sa flottille avec une peinture similaire, créée en mélangeant un gris moyen avec une petite quantité de rouge vénitien.

Groupe Extraction (Base de données)

Objectifs principaux :

Ce groupe s'est chargé de la collecte, du nettoyage, de l'organisation et du stockage des données nécessaires pour l'analyse.

Tâches réalisées :

  • Extraction et nettoyage des données :
    • Analyse initiale du jeu de données provenant de l'étude "Student Alcohol consumption".
    • Nettoyage des données pour éliminer les incohérences ou les données inutiles.
  • Mise en place dans un SGBD :
    • Implémentation des bases de données relationnelles dans PostgreSQL.
    • Définition des relations entre les tables et chargement des données nettoyées.
  • Ajout des nouvelles données sur le temps de jeu vidéo :
    • Mise en place d'un questionnaire afin de récolter des données sur la consommation de jeu vidéo pour les étudiants en étude supérieures.
    • Intégration des données collectées sur les habitudes de jeu vidéo dans la base existante tout en garantissant l’intégrité des relations.

Groupe Analyse

Objectifs principaux :

Étudier les corrélations entre le temps de jeu vidéo, la consommation d’alcool et les performances scolaires, en utilisant des méthodes statistiques.

Tâches réalisées :

  • Exploration des données existantes :
    • Analyse descriptive de l'étude "Student Alcohol consumption".
    • Visualisation pour comprendre les relations entre les variables.
  • Analyse statistique :
    • Calcul des corrélations entre les variables : par exemple, identifier si la consommation d’alcool ou le temps de jeu vidéo influence significativement les performances scolaires.
    • Identification d’effets combinés potentiels (ex. impact simultané du jeu vidéo et de l’alcool).
  • Comparaison :
    • Comparaison des résultats obtenus entre la consommation d’alcool et le temps de jeu vidéo, pour mieux cerner leurs impacts respectifs sur les performances scolaires.

Groupe Interface Web

Objectifs principaux :

Concevoir une plateforme web interactive permettant de visualiser et d’explorer les résultats obtenus par les deux autres groupes.

Tâches réalisées :

  • Conception de l’interface utilisateur :
    • Développement d’une interface intuitive pour afficher les données et analyses, avec une mise en avant des corrélations observées.
  • Intégration de visualisations interactives :
    • Création de graphiques interactifs pour permettre aux utilisateurs d’explorer les résultats.
  • Développement d’un système de navigation :
    • Implémentation d’un menu permettant de naviguer entre les différentes sections (ex. résumé des analyses, graphiques spécifiques, détails des données).
  • Connexion avec la base de données :
    • Mise en place d’API REST pour récupérer les données dynamiques depuis le serveur.
    • Affichage des données en temps réel lorsque la base est mise à jour.
